# Angel Agents — Next.js + Supabase + Vercel Template

Starter template for Angel Agents BUILD stage.

## Stack
- Next.js 15 (App Router)
- Supabase (Auth + Database)
- Tailwind CSS v3
- TypeScript

## Setup
1. Clone and `npm install`
2. Copy `.env.example` → `.env.local`
3. Set Supabase URL + anon key
4. `npm run dev`
